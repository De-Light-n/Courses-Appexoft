# 🧠 Алгоритми та структури даних (бонусний розділ)

Фундамент, який дають в університетах у курсі «Алгоритми та структури даних», зібраний практично й
українською. Це те, що **найчастіше** питають на технічних співбесідах і що відрізняє інженера від
«кодера». Розділ **не пронумерований** — проходьте його паралельно з основними модулями (найкраще
після [Модуля 1](../modul-1-vstup-do-python/README.md)).

## 📓 Матеріали
- **Урок 1:** [Складність та рекурсія](lektsiya-1-skladnist-ta-rekursiya.ipynb) — Big-O/Θ/Ω, класи складності, амортизація, рекурсія, master-теорема
- **Урок 2:** [Базові структури даних](lektsiya-2-bazovi-struktury-danykh.ipynb) — масиви, зв'язані списки, стек, черга, дек, хеш-таблиці
- **Урок 3:** [Дерева, купи, Union-Find](lektsiya-3-dereva-kupy-union-find.ipynb) — обходи, BST, збалансовані дерева, heap, trie, DSU
- **Урок 4:** [Сортування та пошук](lektsiya-4-sortuvannya-ta-poshuk.ipynb) — двійковий пошук, bubble/selection/insertion, merge/quick/heap, counting, Timsort
- **Урок 5:** [Графи](lektsiya-5-grafy.ipynb) — представлення, BFS/DFS, топологічне сортування, Dijkstra, Bellman-Ford, MST
- **Урок 6:** [Парадигми алгоритмів](lektsiya-6-paradygmy-algorytmiv.ipynb) — розділяй і володарюй, жадібні, динамічне програмування, backtracking (+рядки/числа/біти)
- **📝 Домашнє завдання:** [задачі за всіма темами](domashnie-zavdannia-algorytmy.ipynb)

## Що тут є (університетська програма)

### Структури даних
Масиви й динамічні масиви · зв'язані списки (одно/двобічні) · стек · черга · дек ·
хеш-таблиці (колізії, chaining/open addressing) · дерева й BST · збалансовані дерева (AVL, red-black, B-tree) ·
купа / черга з пріоритетом · trie · Union-Find (DSU).

### Алгоритми
Аналіз складності (Big-O/Θ/Ω, best/avg/worst, амортизація) · рекурсія та master-теорема ·
сортування (прості, merge/quick/heap, лінійні) · двійковий пошук ·
графові алгоритми (BFS, DFS, топосорт, Dijkstra, Bellman-Ford, Floyd-Warshall, Kruskal, Prim) ·
парадигми (divide & conquer, greedy, DP, backtracking) · короткий огляд рядкових алгоритмів,
теорії чисел і бітових операцій.

## 🧰 Що встановити (для схем-діаграм)
Уроки 1–5 містять **графічні схеми** (дерева, графи, хеш-таблиці, криві складності), побудовані
через **matplotlib** та **networkx**. Картинки вже **збережені** в зошитах — видно й без запуску.
Щоб самому **перезапускати** ці комірки (`📊 Підготовка візуалізацій`), встановіть бібліотеки:

```bash
pip install matplotlib networkx
```

> Решта коду (структури, алгоритми) працює на **чистому Python** — жодних залежностей не треба.
