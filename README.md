# 🎓 Курс Python: від нуля до Junior+

Практичний курс backend-розробки на Python українською. Мета — не просто «навчити кодити», а
вивести на рівень **Junior+**, здатний **пройти співбесіду**: крім практики, скрізь вплетено
теорію, яку реально питають (позначка **🔎 Важливо знати** та **🎯 питання співбесіди**).

Увесь матеріал — **інтерактивні Jupyter-зошити**: читаєте пояснення, запускаєте код, змінюєте,
експериментуєте.

---

## 🚀 Почни звідси

1. **Встанови інструменти:** [завантаження програм](zavantazhennya-program.md) ·
   [встановлення Git](vstanovlennya-git.md) · [налаштування VS Code](vscode-setup.md).
2. **Навчися здавати домашки** через Pull Request — [CONTRIBUTING.md](CONTRIBUTING.md).
   > ⚠️ Домашки здаються через **Git + PR уже з Модуля 1**. Мінімум для старту (встановити Git,
   > зробити гілку, коміт, PR) є в [CONTRIBUTING.md](CONTRIBUTING.md); повне занурення в Git —
   > у розділі [«Гіт та інше»](git-ta-inshe/README.md) (проходьте паралельно, коли зручно).
3. **Проходь модулі по порядку** (кожен спирається на попередній).
